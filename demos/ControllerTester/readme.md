# Controller Tester
This tool help test the controllers connected to the gamepad ports.

<img src="assets/controller-tester.png" alt="3.3V Voltage Regulator" width="250" >
 
## Features

* Hot-plug detection
* Detects gamepads and the Nintendo SNES mouse
